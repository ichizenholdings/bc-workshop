# NFT Marketplace dApp

シンプルなNFTマーケットプレイスのフロントエンド実装です。ERC721トークンの表示、リスト、購入機能を提供します。

## 機能

- NFTギャラリー表示
- ウォレット接続（MetaMask）
- 所有NFTの表示
- NFTのマーケットプレイスへのリスト
- NFTの購入

## 技術スタック

- Web3.js: イーサリアムブロックチェーンとの対話
- Ethereum Blockies: ウォレットアドレスのアイコン生成
- CSS Variables: テーマカラーの一元管理
- モジュラーJS: 再利用可能なコンポーネント設計
